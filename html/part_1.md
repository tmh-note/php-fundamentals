# HTML Structure

```html
<html>
<head>
    <title>Hello</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>
```

# Element

```
<open tag> ... <close tag>
```
 eg.
```html
<h1> Hello World! </h1>
```

# Attribute

```html
<h1 key=value>
```
 eg.
```html
<h1 title="This is heading"> Hello World! </h1>
```
